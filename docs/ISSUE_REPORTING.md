# Issue Reporting

Thanks for opening an issue for mtab.

Before submitting, please quickly check for existing issues so we can avoid duplicates.

If relevant for your issue, include:

- what happened
- what you expected
- steps to reproduce
- browser and version
- mtab version
- operating system
- screenshots or recordings
- console errors or logs
- any custom CSS, fonts, wallpapers, or shared configs involved

For feature requests, it helps to explain the problem you want solved and any idea you have for improving it.
